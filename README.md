=====> FUTRAN SOLUTIONS React Js Template Project

# Purpose :-

This project is a modern React JS + TypeScript frontend template, designed for rapid development with best practices in code quality, testing, and build tooling. It is configured with Vite for Fast Builds, ESLint for Linting, Prettier for formatting, Vitest for Unit Testing with Strict rules of Typescript ,and Eslint.

# Includes :-

- React + TS ready-to-use project
- Strict linting, type & formatting rules
- Vite Unit testing
- Reusable UI components with Material UI design library.
- .css file for writing Css.
- Pnpm package manager
- For routing, uses react-router-dom

# Start Development Server => pnpm dev

# Installation => pnpm install

# Run tests => pnpm test
